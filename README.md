# No Filter

To start the app you need to first install the app loader on your device:

```
npm run install:android
```

Then you need to launch the app server:

```
npm start
```

## Dev Features

  - Hot reloading app
  - Auto reloading server
  - Flow typechecking
  - Eslint linting
